大概意思就是parameter是函数定义好的参数接口，而argument是实际上传入函数的参数，

git

1.git rm --cached filename
从本地缓存区移除filename这个文件
git rm --cached filename -f
强制执行从本地缓存区移除filename这个文件
2.git fetch
将远程仓库的内容拉取到本地，并且创建一个本地的分支叫origin/master
3.git merge branch
将branch的内容合并到当前分支，生成一个新的节点，有内容冲突会在文件中自动生成，可手动取舍
4.git pull
pull=fetch+merge
5.git checkout branch
切换到branch分支，本地工作文件全部变为新的head指向的内容

三次握手的小感悟

Posted on 2022-02-22 | In Computer Network

三次握手的小感悟

核心在于，任何连接都必须建立在双方知情的基础之上，否则就会产生无法释放的死连接。
例如第二次握手，如果client没有收到，那么就会尝试再次发送请求，
如果server不经过第三次握手就建立了连接，那么第一次尝试的请求就成了client不知情的请求。
一旦这个连接双方都知情，那么后续处理就不需要担忧了。

如果出现错误，可以通过四次挥手来终止这次连接。
同样四次挥手需要确保双方知情。
A发出结束请求，
B表示同意，
那么A就可以大胆地停止传输，等待B结束
B发出结束请求
A表示同意，
结束

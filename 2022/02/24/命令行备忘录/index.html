命令行备忘录

Posted on 2022-02-24 | In tips

从今天开始记录一些遇到的命令行，方便以后查

git

1.git rm --cached filename
从本地缓存区移除filename这个文件
git rm --cached filename -f
强制执行从本地缓存区移除filename这个文件
2.git fetch
将远程仓库的内容拉取到本地，并且创建一个本地的分支叫origin/master
3.git merge branch
将branch的内容合并到当前分支，生成一个新的节点，有内容冲突会在文件中自动生成，可手动取舍
4.git pull
pull=fetch+merge
5.git checkout branch
切换到branch分支，本地工作文件全部变为新的head指向的内容
6.

正常来说，将本地的更新上传到远程仓库用上面这一套命令就足够了
可以配置.gitignore的内容来屏蔽一些不想上传的文件

conda

1.conda info
查看conda整体信息
2.conda info --env/conda info -e
查看conda当前选择的虚拟环境
